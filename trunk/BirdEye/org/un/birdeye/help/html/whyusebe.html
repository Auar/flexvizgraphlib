Why Use BirdEye?

Concepts:

BirdEye is a Link Analysis tool.  Link analysis is a method of exploring associations between objects. Link analysis provides visibility of the relationships and associations between objects of different types that are not apparent from isolated pieces of information.

The basic steps and usage of BirdEye are as follows:

Adding Data

1.)  Create "nodes" which may be persons, places, things, events, or any other object of interest.  Use the data entry interface to indentify and describe graph objects.
2.)  Create links or "edges" between nodes based on an association type such as friend, associate, member of, etc.

Objects created through the data entry interface are stored in a SQLite database.

Visualization

    • A number of common graph layouts are provided to enable the best fit for a particular data set.  Each layout includes a number of controls to "tweak" the parameters of the algorithm.
    • Hierarchical Node Spacing:  controls the spacing between siblings which when used in conjunction with Sibling Spread can help optimize a Hierarchical graph.
    • Layout angle is used in both the Parent-Centered and Phyllo layouts to control the geometry of the layout.
    • Several controls are provided which are applicable to all layouts:
      1. Degrees of Separation - used to control the distance of the relationships you want to view, e.g. friend of friend of friend.
      2. Scaling and Zoom are helpful for large graphs where changing node size or magnifying the canvas space is needed.
      3. Autofit will automatically plot the graph to within the bounds of the view space.
      4. History will show breadcrumbs of nodes visited.
    • Refresh!  Important - refresh will make user initiated calls to the database to load new data.  This does not happen automatically so as not to destroy works in progress.
